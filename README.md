# 储能投资分析系统

> Energy Storage Investment Analysis System

一个专业的储能项目财务分析与投资决策工具，支持全生命周期财务建模、IRR/NPV计算、多维度敏感性分析。

## 功能特性

### 核心功能

- **投资分析** - 静态/动态投资计算、资本金与贷款配置、建设期利息计算
- **财务指标** - 全投资IRR（税前/税后）、资本金IRR、NPV、静态投资回收期
- **贷款管理** - 等额本息/等额本金还款计划、利率变更、提前还款模拟
- **成本核算** - 折旧计算、运维成本、人工成本、保险费用、土地费用
- **利润分析** - 营业收入、增值税计算、所得税计算、利润分配
- **现金流分析** - 全投资现金流、资本金现金流、累计现金流

### 可视化图表

- 投资构成饼图
- 资金来源饼图
- 收入趋势图
- 成本构成图
- 现金流趋势图
- IRR敏感性分析图
- 投资回收期图
- 还本付息计划图

### 数据导出

- Excel报表导出（完整财务数据）
- PDF报告生成

## 核心计算模块

### 投资计算

- 静态总投资 = 设备购置费 + 建筑工程费 + 安装工程费 + 其他费用
- 动态总投资 = 静态投资 + 建设期利息
- 资本金 = 动态投资 × 资本金比例
- 长期借款 = 动态投资 - 资本金

### 财务指标

- **IRR** - 内部收益率，采用牛顿迭代法求解
- **NPV** - 净现值，默认折现率8%
- **投资回收期** - 累计现金流由负转正的时点

### 税务政策支持

- 西部大开发优惠（所得税15%）
- 高新技术企业优惠（所得税15%）
- 增值税即征即退

## 验证基准

系统以**邛崃100MW/200MWh储能项目**为验证基准：

| 指标 | 预期值 |
|------|--------|
| 静态投资 | 15,846.26 万元 |
| 动态投资 | 16,036.41 万元 |
| 全投资IRR（税前） | 6.38% |
| 资本金IRR | 17.97% |
| 投资回收期 | 9.20 年 |
| 年均营业收入 | 3,175.15 万元 |
